<h2> Descrição :pencil2: </h2>
<p>
  Site focado no ramo automobilístico, fornecendo ao usuário informações interessantes e confiáveis do mundo automotivo.<br>
  O grupo Web-motores fornece matérias com o intuito de dar a usuário informações relevantes e conhecimentos indispensáveis para os amantes do mundo sobre rodas.<br>
  No findar da página home há a opção de cadastrar-se no banco de dados da empresa, afim de receber novas notícias do meio automotivo.
</p>

<h2> Funcionalidades :bulb: </h2>

- `Funcionalidade 1` : Design responsivo usando o conceito mobile first.
- `Funcionalidade 2` : Validação de formulário com Javascript.
- `Funcionalidade 3` : Uso de localStorage para armazenamento dos dados do formulário.
- `Funcionalidade 4` : Busca dos dados armazenados no localStorage para evitar cadastros com dados repetidos.
